PCS 15 /Digital Security

The text in this section is used under CC-BY license from Progressive, Accelerated Certifications for Employment in Information Technology (PACE-IT) TAACCCT Grant, retrieved from www.skillscommons.org.

There are many areas where digital security measures should be implemented. Security measures should reside at both the computer level and at the network equipment level. The measures that get enacted will be different for each.

Antivirus

One important area of digital security involves antivirus applications. Antivirus software should be installed, active, and up-to-date on every machine. If it's not installed, it can't stop a virus. If it's not active, it can't stop a virus. If it's not up-to-date, it won't recognize a virus.

Highlights:

● Antivirus software should be installed, active, and up-to-date on every machine.

● Make sure your antivirus software is able to do its job effectively.

Antispyware

Antispyware is closely related to antivirus. As a matter of fact, a lot of antivirus software packages have antispyware functionality built into them. Spyware is malicious code that collects information about a system. It may change some system settings, but what it mainly does is to collect information and then send that information off to a remote site. Antispyware can prevent the malicious code from running. It can also assess a system, help the user determine if spyware is present, and evaluate the level of threat that the spyware represents.

Highlights:

● Spyware is malicious code that collects information about the system and may change some settings.

● Antispyware can prevent that code from running. It can also access a system and help an administrator or a user determine if spyware is present and what level of threat it represents.

Firewalls

A discussion on digital security would not be complete without including information on firewalls. Software firewalls should be installed and active on every machine. The firewalls on computers tend to be simpler than those that are installed on routers. Firewalls are the traffic cops of your network traffic, controlling the flow of data into and out of the network and the PCs that are attached to it. Firewalls help prevent attacks — or mitigate attacks once they occur — by recognizing suspicious activity. Make sure that you implement a good-quality software firewall on every device.

Highlights:

● Software firewalls should be installed and active on every machine.

● Firewalls are the traffic cops of network traffic, controlling the flow of data into and out of the network and preventing malicious actions from occurring.

User Authentication

User authentication is also extremely important when it comes to digital security. There are three main ways to authenticate a user. There's the “what you know” method — such as username and password; the “what you have” method — such as a security token; and the “what you are” (biometrics) method — such as fingerprint and retinal scans. You can implement a combination of these factors to increase your security level.

Highlights:

● User authentication can be implemented using three main methods: what you know, what you have, and what you are.

● Combine the factors for increased security.

Password Strength

Closely related to authentication, strong passwords are very important. The strongest passwords are random strings of letters, numbers, and symbols. But they are kind of hard to remember. The weakest passwords are whole words, consecutive numbers, or easily guessed passwords — like a birth date or an anniversary.
