PCS 16 /數據銷毀與處置

總覽

在本模塊中，我們將介紹銷毀數據和硬盤驅動器的不同方法。我們還將討論正確覆蓋硬盤的方法。

低級格式與標準格式

我們大多數人都熟悉格式化磁盤驅動器。一個普遍的誤解是格式化磁盤驅動器會破壞數據。格式化磁盤的主要目的是為新的操作系統安裝做好準備，並且為了使其生效，磁盤上當前的任何數據都需要通過內部指針擦除或設置，以使數據佔據這些扇區。重新格式化後保存並重寫。

標準格式過程或高級格式可以通過兩種方式完成-快速格式和完整格式。快速格式可以使磁盤準備好接收文件結構，例如FAT32或NTFS，而完整格式還可以檢查驅動器中的壞扇區並對其進行標記。自Windows Vista以來的所有Windows版本中，完整格式還將零和一寫入分區。

快速格式並沒有真正破壞數據，它只是刪除了文件結構的索引，這使得查找文件稍微困難一些。但是，數據並沒有消失，並且有許多可用的軟件工具可以從經過快速格式化的磁盤中恢復文件。但是，有一些工具可以在格式化過程中將隨機零和一寫入磁盤，這是一種有效的數據銷毀方法。

低級格式化是不同的，它是銷毀數據的有效手段，但在製造硬盤驅動器後無法真正完成。由於現代ATA / IDE驅動器的特性，磁盤的幾何形狀因製造商而異，真正的低級格式化僅在製造過程中發生，並且在用戶或技術人員將磁盤投入使用後無法重做。這是因為低級格式實際上定義了驅動器上磁道和扇區的位置。但是，主要的HDD製造商確實提供了實用程序，這些實用程序將重新初始化磁盤，測試其壞扇區，將零和一個寫入扇區以及重新分區自己的磁盤。這是技術人員可以進行的最底層格式化。

標準格式從操作系統內部發生。例如，技術人員可以在Windows File Explorer中連接新的外部硬盤，然後右鍵單擊該磁盤，然後選擇Format，如下圖所示。然後，它將執行新驅動器的標準格式。

Windows File Explorer彈出菜單，帶有格式化選項。Jason Dion在CC-BY許可下使用的圖像。

如果驅動器使用FAT文件系統格式化，則此過程將擦除文件分配表（FAT），如果文件系統為NTFS（新技術文件系統），則此過程將擦除文件指針。這不會擦除驅動器上的數據，只會擦除通過常規操作系統訪問數據的方法。技術人員仍然可以使用為此目的而構建的軟件工具來恢復數據。

例

一個很好的類比是，如果您去教科書並刪除了第17章的目錄條目。您仍然可以逐頁翻閱教科書，直到找到第17章為止，但您將不再知道第17章由於指針（目錄項的表）已被刪除，所以它位於頁672上。

強調

  • 在Windows Vista之前的操作系統中，快速和完整格式方法都僅準備磁盤以接收文件結構。完整格式還會檢查驅動器中的壞扇區並對其進行標記。
  • 從Windows Vista開始，完整格式還會將零和一寫入磁盤，從而幾乎不可能進行數據恢復。
  • 標準格式不會破壞文件，只會破壞文件結構的索引。
  • 數據恢復工具可用於檢索數據。
  • 硬盤數據消滅方法

共有三種執行HDD數據銷毀的方法。他們是：

  • 清算
  • 消毒或清除
  • 物理破壞

如果要將磁盤重新用於公司內的另一個工作站，或者將該工作站捐贈給另一個組織，則在不破壞硬盤的情況下破壞數據很重要。為了完成此任務，技術人員可以使用HDD製造商的軟件執行低級格式化，並確保其至少一次將零和一寫入磁盤。這稱為 清算。

市場上有許多符合或超過美國國防部5220.22-M標準的程序。這指的是1997年國防部國家工業安全計劃操作手冊（NISPOM）指南，要求政府承包商遵循確切的程序處理需要清理的機密數據 。七次重寫已成為計算機行業採用的事實上的標準，成為回收HDD並使它們保持使用狀態的有效手段。

彼得·古特曼（Peter Gutmann） 將該標準提高到35次覆蓋。在此過程中，硬盤的每個位都被零或一個或隨機字符覆蓋。到達磁盤末尾後，程序將從頭開始再次啟動，直到整個磁盤至少被覆蓋35次為止。但是，對於大多數用途而言，七次足以防止數據恢復。

完成此操作的程序通常稱為磁盤擦除，磁盤清理或驅動器擦除程序。請注意，這些方法要花費大量時間，尤其是在硬盤容量較大時，因為每個位都必須被重寫多次。

由於人們對安全性的日益提高的意識以及HDD存儲設計的不斷變化的性質，該行業正在遠離已不再公開的DOD 5220.22-M標準。根據布賴恩·史密森（Brian Smithson）在Infosec島上的一篇文章，當前要遵循的標準是美國國家標準技術研究院的“ NIST特別出版物800-88”。當前的研究正在引導業界使用此標準中確定的“安全擦除”技術，並發現“單遍覆蓋”就足夠了。

永久清除磁盤上數據的另一種方法是使用電磁體對硬盤進行消磁（消磁）。使磁盤消磁的系統非常昂貴，並且僅在非常安全的環境中使用。一些公司通過按服務付費模式提供硬盤驅動器處置。通常，與購買昂貴的設備相比，小型企業為處置服務付費更經濟。

處理CD，DVD或軟盤時，最好的銷毀方法是物理銷毀，因為這些介質的更換成本非常低廉。銷毀這些物品最簡單的方法是專門為該任務而建造的碎紙機，通常價值約50至100美元。

儘管快速擦除數據可以防止普通計算機用戶訪問數據，但是大多數技術人員或黑客仍可以檢索到刪除過程後留下的剩餘數據，稱為數據保留。數據恢復工具變得越來越容易使用，更擅長恢復數據，並且更易於使用，這使得恢復已刪除文件的工作在許多用戶的範圍之內。因此，使數據不可訪問的最佳方法是物理破壞硬盤驅動器。這可以通過許多不同的方法來完成，包括使用錘子，鑽頭或其他機械設備破壞盤片，使驅動器僅可作為廢金屬回收利用。銷毀是永久刪除數據的最安全方法。

選擇正確的方法進行安全的數據銷毀

當時間更受關注，而數選擇正確的方法進行安全的數據銷毀據安全性不那麼受關注時，可以採用快速格式化。但是，由於熟練的技術人員或黑客可以在快速格式化後恢復數據，因此當公司內部重新使用計算機時，最好進行完整格式化。

如果您或您的公司丟棄了計算機，則可能會擔心其他人可能訪問敏感或機密信息。因此，快速格式化不是最佳選擇，即使完整格式可能就足夠了，但要確保任何人都無法訪問任何舊數據的責任和責任可能需要物理破壞，例如在硬盤驅動器盤片上鑽孔，粉碎用錘子敲擊驅動器，或燒毀驅動器。物理破壞是防止數據恢復的最佳永久解決方案。

如果您或您的公司正在出售計算機或將計算機轉讓給另一個組織，則無法進行物理銷毀，並且快速格式化還不夠。在這種情況下，應按照DOD 5220.22-M標準擦除磁盤，方法是用0和1的每一位覆蓋磁盤七到35次，或使用安全擦除，如“ NIST特殊出版物800-88”中所述。

有關免費軟件Secure Erase和Gordan Hughes博士在該主題上的工作的更多信息，請參見此處。加州聖地亞哥聖地亞哥分校提供的安全擦除軟件可在此處獲得。此處提供了三種處置方法的摘要。

使用Windows工具擦除和格式化硬盤

如前所述，在您要保留計算機或將其重新使用以在同一公司內使用的情況下，Windows工具提供的完整格式就足夠了。

接下來，我們將選擇我們要在驅動器上使用的文件系統類型。文件系統只是數據在卷上存儲和訪問的方式。有幾種不同的文件系統。從Windows NT 4.0開始，NTFS是所有Microsoft操作系統使用的本機文件系統，它取代了Windows 95和Windows 98使用的FAT32文件系統。WindowsVista和更高版本的任何啟動卷都需要NTFS。FAT32是較舊的文件系統類型，與NTFS不兼容且不如NTFS安全，但是可以由NTFS操作系統讀取和寫入。儘管Apple使用HFS和HFS +文件系統，但是Apple計算機可以讀取和寫入FAT32。但是，FAT32具有大小限制。它不能在大於32 GB的磁盤上使用，也不能存儲大於4 GB的文件。

在我們的示例中，我們將接受默認的NTFS。我們還可以在此處選擇執行快速格式化（默認情況下選擇）還是執行完整格式（通過取消選中快速格式化選項）。我們將做一個快速格式化。
